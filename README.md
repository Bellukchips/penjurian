## Create Database
```bash
penjurian
```
## Migrate All Table
```bash
php artisan migrate
```
## Use Database Seeder For Admin
```bash
php artisan db:seed
```
## Running
```bash
php artisan serve
```
## Another way to run the app
launch url
```bash
    penjurian.test
```
